| 파일명 | 내용 | 수량 |

## 3. 권장 프린트 설정

| 항목 | 값 | 비고 |
|---|---|---|
| **재료** | PLA 또는 PETG | PLA 권장 (칫수 안정성) |
| **레이어 높이** | **0.12 mm** 이하 | 0.2mm도 가능하나 정밀도 낮아짐 |
| **노즐** | 0.4 mm 이하 | 0.2mm 면 더 좋음 |
| **벽 라인 수** | 3줄 이상 | 벽이 2.4mm라 여유있음 (0.4mm 노즐 기준 6줄 가능) |
| **인필** | 30-50% | 작아서 큰 차이 없음 |
| **상/하면 레이어** | 4-5 레이어 | |
| **서포트** | **불필요** (아래 방향 참고) | |

## 4. 프린트 방향 (중요!!)

두 파트 모두 **열린 면이 빌드 플레이트를 향하도록** 배치해주세요. STL은 이미 이 방향으로 준비되어 있습니다.

- **bottom_case.stl**: 케이스 바닥면이 아래로, 벽이 위로 솟는 형태. 자연스럽게 이 방향으로 슬라이서에 로드됨.
- **top_case.stl**: 뚜껑 외부면이 아래로, 내부 공동이 위를 향함. 이미 뒤집어서 export 했으니 그대로 프린트.

이 방향이면 **서포트 없이** 깔끔하게 뽑힙니다.

> USB-C 개구부의 천장 부분(8.6mm 폭의 좁은 오버행)만 슬라이서가 서포트를 추천할 수 있는데, 8.6mm 정도는 **브리지(bridge)로 처리 가능**하니 서포트 끄셔도 됩니다.

---

## 5. 공차 주의사항 ⚠️ (스냅핏이라 중요)

- **상·하단은 0.2mm 간섭 압입(press-fit)** 으로 설계되어 있습니다. 접착제 없이 끼워 맞추는 구조예요.
- 프린터 공차가 **±0.1mm 이내**여야 제대로 맞습니다. XY 칫수가 너무 크게 뽑히면 **압입이 너무 타이트해서 안 닫힐 수 있고**, 너무 작게 뽑히면 헐거워집니다.
- **가능하면 테스트 큐브로 XY 스케일 한번 체크**해 주시면 감사하겠습니다. 필요시 수평 공차(horizontal expansion) 슬라이서 설정으로 ±0.05mm 보정 가능.

---

## 6. 후가공

- **USB-C 개구부**는 실제 보드의 포트가 정확히 들어맞도록 설계되어 있습니다. 프린트 직후 포트가 안 들어가면 사포(#400~600)로 살짝 다듬어 주세요.
- **스냅핏 가장자리 (립/그루브)** 의 버는 칼로 깔끔히 제거 부탁드립니다. 버가 남아있으면 조립이 어려워져요.

---

## 7. 최종 조립 확인 요청

프린트 후 **실제로 상단과 하단이 맞물리는지 1회 체결 테스트** 부탁드립니다.
- 너무 안 닫히면 → 수평 공차 −0.05mm 적용해서 재출력
- 너무 헐거우면 (손으로 쉽게 분리됨) → 수평 공차 +0.05mm 적용해서 재출력
